# 🎬 Netflix Data Analysis Project

## 📌 Project Overview
This project explores the **Netflix Movies and TV Shows Dataset** from Kaggle using Python.  
The goal is to perform **Exploratory Data Analysis (EDA)** and discover trends in Netflix content.  

## 📂 Dataset
- Source: [Netflix Titles Dataset on Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)  
- Format: CSV file (`netflix_titles.csv`)  

## 🛠️ Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

## 🔍 Key Analysis & Insights
- Distribution of **Movies vs TV Shows**  
- Top **countries** producing Netflix content  
- Content trends by **release year**  
- Most frequent **genres/categories**  
- Insights into **ratings**  

## 📊 Visualizations
- Bar charts showing content distribution  
- Heatmap of correlations  
- Trends of Netflix content over time
